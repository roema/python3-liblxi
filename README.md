# python-liblxi (Python3)

Fork from https://github.com/lxi-tools/python-liblxi


## build liblxi

```
$ cd python3-liblxi/liblxi
$ ./autogen.sh
$ ./configure
$ make
$ make install
```

## cmake python

```
$ mkdir bld
$ cd bld
$ cmake ..
$ make
$ sudo make install
```
